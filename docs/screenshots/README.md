# スクリーンショット & デモ動画

このディレクトリには、DocuFlow の UI を紹介するためのスクリーンショットとデモ動画を保存します。

## 必要なファイル

以下のスクリーンショットを追加してください：

### 必須

- ✅ `dashboard.png` - ダッシュボード画面（`/app`）
  
  **撮影時の推奨内容**:
  - 左サイドバーが表示されている状態
  - 概要カード（ドキュメント総数・ピン留め・お気に入り・インサイト）が表示されている
  - カテゴリ別トップ3ミニグラフが表示されている
  - 詳細検索フォーム（日付範囲・共有中フィルタ含む）が表示されている
  - 複数のドキュメントカードが表示されている（タグ・カテゴリ・お気に入り・ピン留めが付いたもの）
  - 最近のアクティビティセクションが表示されている
  - ショートカットヘルプボタン（？）が表示されている
- ❌ `new-document.png` - 新規ドキュメント作成画面（`/new`）
- ❌ `document-detail.png` - ドキュメント詳細画面（`/documents/[id]`）
- ❌ `settings.png` - 設定画面（`/settings`）
- ❌ `share-view.png` - 共有リンク閲覧画面（`/share/[token]`）

### オプション（推奨）

- `demo.gif` または `demo.mp4` - デモ動画（ドキュメント作成から共有までの流れ）

## 撮影のコツ

1. **画面サイズ**: 1920x1080 または 1440x900 を推奨
2. **ブラウザ**: Chrome または Safari の最新版を使用
3. **データ**: 実際のデータ（ドキュメント、タグ、カテゴリなど）が表示されている状態で撮影
4. **プライバシー**: 本番環境で撮影する場合は、機密情報が含まれていないことを確認

## 動画の作成方法

### GIF の作成（macOS）

1. [Kap](https://getkap.co/) や [Gifox](https://gifox.io/) などのツールを使用
2. または、動画を撮影して [CloudConvert](https://cloudconvert.com/) などで GIF に変換

### GIF の作成（Windows）

1. [ScreenToGif](https://www.screentogif.com/) などのツールを使用
2. または、動画を撮影してオンラインツールで GIF に変換

### 推奨デモ内容

1. ログイン
2. ダッシュボードの表示
3. 新規ドキュメント作成（PDF アップロード）
4. AI 要約の自動生成
5. ドキュメント詳細の表示
6. 共有リンクの発行
7. 設定画面の表示

## ファイルサイズ

- **画像**: 各 PNG ファイルは 500KB 以下を推奨（GitHub の表示速度を考慮）
- **動画**: GIF は 5MB 以下、MP4 は 10MB 以下を推奨

## 注意事項

- スクリーンショットに個人情報や機密情報が含まれていないことを確認してください
- 本番環境で撮影する場合は、テストデータのみを使用してください
