# ダッシュボード画面のスクリーンショット撮影ガイド

`dashboard.png` を更新する際の、推奨される撮影内容と手順です。

## 撮影前に準備するデータ

以下のようなテストデータがあると、より良いスクリーンショットになります：

1. **ドキュメント**（5-10件程度）
   - 異なるカテゴリ（仕様書、議事録、企画書など）
   - タグが付いたもの
   - ピン留めされたもの
   - お気に入りに追加されたもの
   - 共有リンクが発行されているもの

2. **アクティビティログ**（直近10件）
   - ドキュメント作成、更新、削除などの履歴

## 撮影手順

### 1. ブラウザの準備

1. Chrome または Safari を開く
2. 開発者ツール（F12 または Cmd+Option+I）を開く
3. デバイスツールバーを有効化（Cmd+Shift+M）
4. 画面サイズを **1920x1080** または **1440x900** に設定

### 2. ダッシュボード画面を開く

1. `/app` にアクセス
2. ログイン済みの状態でダッシュボードを表示

### 3. 表示内容の確認

以下の要素がすべて表示されていることを確認：

- ✅ **左サイドバー**
  - Logo（DocuFlow）
  - 「ドキュメント」リンク（アクティブ状態）
  - 「新規作成」リンク
  - 「設定」リンク
  - 「ログアウト」リンク

- ✅ **ヘッダー**
  - 「ドキュメントワークスペース」タイトル
  - 合計件数・ピン件数・お気に入り件数
  - ショートカットヘルプボタン（？）
  - ユーザーメニュー

- ✅ **概要カード（4枚）**
  - ドキュメント総数カード（グラデーション背景）
  - ピン留めカード
  - お気に入りカード
  - インサイトカード（直近30日件数・カテゴリ数・共有リンク数・閲覧回数・平均文字数・アーカイブ数・4週間グラフ）

- ✅ **カテゴリ別トップ3ミニグラフ**
  - カテゴリ名と件数のバーグラフ

- ✅ **ドラッグ＆ドロップアップロードエリア**
  - 「ファイルをドラッグ＆ドロップ」のメッセージ

- ✅ **詳細検索フォーム**
  - 検索入力欄
  - カテゴリセレクト
  - 並び順セレクト（新しい順/古い順/ピン留め優先）
  - 作成日（From/To）
  - チェックボックス（ピンのみ/お気に入りのみ/共有中のみ）
  - 検索ボタン・新規作成ボタン
  - クイックフィルタリンク

- ✅ **ドキュメントカード一覧**
  - 複数のカード（2列グリッド）
  - 各カードに以下が表示されている：
    - タイトル
    - カテゴリバッジ
    - 作成日時
    - アーカイブバッジ（該当する場合）
    - 共有中バッジ（該当する場合）
    - ピン留め・お気に入り・削除ボタン
    - 要約（4行まで）
    - タグ（複数）
    - 文字数・コメント数
    - アーカイブ・削除ボタン

- ✅ **最近のアクティビティセクション**
  - 「最近のアクティビティ」タイトル
  - アクティビティリスト（直近10件）

### 4. スクロール位置の調整

- ページの上部から表示されるようにスクロール
- 概要カードとカテゴリ別グラフが完全に表示されている状態

### 5. スクリーンショットの撮影

1. **macOS**:
   - `Cmd + Shift + 4` を押す
   - カーソルで範囲を選択（画面全体をカバー）
   - または `Cmd + Shift + 3` で画面全体を撮影

2. **Windows**:
   - `Win + Shift + S` を押す
   - 範囲を選択
   - または `Win + PrintScreen` で画面全体を撮影

3. **ブラウザ拡張機能**:
   - [Full Page Screen Capture](https://chrome.google.com/webstore/detail/full-page-screen-capture/fdpohaocaechififmbbbbbknoalclacl) などの拡張機能を使用

### 6. ファイルの保存

1. 撮影した画像を `docs/screenshots/dashboard.png` として保存
2. 既存の `dashboard.png` を上書き

## 画像の最適化（オプション）

- **ファイルサイズ**: 500KB 以下を推奨
- **形式**: PNG（透過が必要な場合）または JPEG（ファイルサイズを抑えたい場合）
- **解像度**: 1920x1080 または 1440x900

画像最適化ツール:
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)

## チェックリスト

撮影前に以下を確認：

- [ ] 左サイドバーが表示されている
- [ ] 概要カードが4枚すべて表示されている
- [ ] インサイトカードに4週間グラフが表示されている
- [ ] カテゴリ別トップ3グラフが表示されている
- [ ] 詳細検索フォームが表示されている
- [ ] 複数のドキュメントカードが表示されている
- [ ] タグ・カテゴリ・お気に入り・ピン留めが視覚的に分かる
- [ ] 最近のアクティビティセクションが表示されている
- [ ] 画面サイズが適切（1920x1080 または 1440x900）
- [ ] ファイルサイズが 500KB 以下
